# 注意事項

這個 Demo 使用

`ruby 2.2.0`

`rails 4.2.3`

# 事前準備

1. 安裝 redis 資料庫，`$ brew install redis`
2. 啟動 redis 資料庫，`$ brew services start redis`
3. 若需要其他資訊可以輸入，`$ brew info redis`

# 使用 sidekiq

官方網站：
[mperham/sidekiq: Simple, efficient background processing for Ruby](https://github.com/mperham/sidekiq)

1. `gem 'sidekiq'`
2. `$ bundle install`
